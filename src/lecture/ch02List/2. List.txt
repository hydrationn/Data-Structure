Part 2. List
: 일련의 동일한 타입의 항목(Item)들이 나열된 것

- Array
- Singly Linked List
- Doubly Linked List
- Circular Linked List

1. 최악의 경우 수행시간
(1) Array: 접근 O(1), 탐색 O(n), 삽입 O(n), 삭제 O(n)
- 정렬된 배열에서의 탐색은 O(log n)

=> List의 경우, 삽입/삭제될 노드의 이전 노드의 레퍼런스가 주어진 경우에 해당
(2) Singly Linked List: 접근 O(n), 탐색 O(n), 삽입 O(1), 삭제 O(1)
(3) Doubly Linked List: 접근 O(n), 탐색 O(n), 삽입 O(1), 삭제 O(1)
(4) Circular Linked List: 접근 O(n), 탐색 O(n), 삽입 O(1), 삭제 O(1)
